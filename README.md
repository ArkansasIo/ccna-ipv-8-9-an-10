# CCNA Programming in C++23 (CCNA-style labs)

This folder contains **C++23 source code** and **documentation/lab guides** built around common CCNA networking concepts, expressed as programming exercises.

## What’s inside
- `docs/` — step-by-step theory + lab handouts (CCNA topics translated into coding tasks)
- `src/` — reusable C++23 library code (IP parsing, CIDR math, routing-table models)
- `labs/` — complete lab projects (each with a `README.md` and starter code)
- `projects/` — larger optional projects (CLI tools, simulators)

## Build / run (common)

Most examples are header-only or use a simple `g++` compile command.

If you want a unified build system, install/use CMake and update the labs’ `CMakeLists.txt`.

## Notes
- These materials are **educational**: they model network concepts and are not a substitute for Cisco exams/study.
