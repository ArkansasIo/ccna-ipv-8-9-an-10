# Getting Started (CCNA Programming in C++23)

This program-oriented CCNA study material is organized as:
- **Concept docs**: short theory + how it maps to code
- **Lab guides**: exercises with expected outputs
- **C++23 starter code**: small programs you can compile quickly

## Recommended workflow
1. Read the concept doc.
2. Complete the lab exercise by editing the provided C++ files.
3. Compile and run.
4. Compare your output with the expected results.

## Toolchain
Any modern compiler supporting **C++23**:
- GCC/Clang (Linux/macOS)
- MSVC (Windows)

## Common compile command (GCC/Clang)
```bash
g++ -std=c++23 -O2 -Wall -Wextra -pedantic your_file.cpp -o your_file
./your_file
```

## Submission format
For each lab:
- Commit/paste your edited `.cpp` files
- Include a short note: `What I changed` + `What output I got`
